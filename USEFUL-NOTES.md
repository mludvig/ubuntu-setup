# Disable _Windows Terminal_ Ctrl+V intercept

When using `vim` in _WSL2_ in Windows Terminal we can't use _Ctrl+V_ for vertical blocks. 

- Option 1 - Use _Ctrl+Q_ instead (vim alias for _Ctrl+V_)
- Option 2 - Open terminal settings (Ctrl+Shift+,) and delete 'ctrl+v' block from the `keybindings` block.
